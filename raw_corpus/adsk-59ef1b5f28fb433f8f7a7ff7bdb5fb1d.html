Learn about ShotGrid’s basic capabilities and functionality in this introductory course. Set up your account, gain an understanding of the structure of data within ShotGrid, learn to navigate ShotGrid, determine your role, including what you can and cannot do, and customize the view of on-screen data.

After completing this course, you should be able to:

  • Take a tour of ShotGrid and sign up for a free trial.
  • Update your ShotGrid profile photo and account information.
  • Get notifications via the ShotGrid Inbox and email.
  • Explain the ShotGrid schema and common project structures.
  • Navigate ShotGrid’s global menu.
  • Navigate the project navigation menu.
  • Understand ShotGrid’s default permission roles.
  • Learn about ShotGrid’s pages.
  • Navigate Entity list pages.
  • Navigate Entity detail pages.
  • Organize data on an Entity page.
  • Format data on an Entity page.

 

Before you start

  • If you have technical difficulties, please reach out to product support.

About the author 

This content was created by Jessica Parsons, Technical Account Specialist (M&E/ShotGrid) at Autodesk.

Course modules

~20 min.
Take a tour of ShotGrid and sign up for a free trial
Module

Welcome to ShotGrid! If you’re a new user, perhaps you’re wondering – what is ShotGrid? What can it do? Why should I use it? Learn all about the capabilities of Autodesk’s production management software and how it can benefit you in this quick introduction. Learn how to download a free trial of ShotGrid to follow along with this introductory course.

‌
~20 min.
Update your ShotGrid profile photo and account information
Module

Set yourself up for success by ensuring your account and user information is accurate. Make it easier for other users to find you by adding a profile picture. Choose where you start in ShotGrid by setting a homepage.

‌
~20 min.
Get notifications via the ShotGrid Inbox and email
Module

Stay up to date on key information by setting up notifications whenever new Notes, Versions, Publishes, or other important changes are made on anything related to a user’s work. Set up notifications to be sent to your ShotGrid’s inbox and/or to your email inbox.

‌
~15 min.
Explain the ShotGrid schema and common project structures
Module

To use ShotGrid effectively, it is important to understand how its data is structured. Shots, sequences, episodes, tasks, versions, time logs, and more are all considered entities within ShotGrid. The ShotGrid schema refers to how these entities are used and configured and how they are connected to one another.

‌
~45 min.
Navigate ShotGrid’s global menu
Module

Learning to efficiently navigate through ShotGrid is critical to being able to work effectively. This content covers navigation of ShotGrid’s global menu.

‌
~25 min.
Navigate ShotGrid’s project menus
Module

Learning to efficiently navigate through ShotGrid is critical to being able to work effectively. This content covers navigation of ShotGrid’s project menus and customization of project menus to better suit a project’s needs.

‌
~25 min.
Understand ShotGrid’s default permission roles
Module

ShotGrid allows administrators to control who can see and edit different types of information. Control is set by assigning users to a permission role. Learn about the default permission roles within ShotGrid, how to determine your own permission role, and the difference between types of permissions.

‌
~45 min.
Learn about ShotGrid's pages
Module

ShotGrid provides many ways to see the same information, to cater to the needs of each user. Learn about page types, page access, modifying and saving pages, and keeping pages organized.

‌
~25 min.
Navigate Entity list pages
Module

List pages allow users to see information about a list of entity records, such as a list of Assets or Tasks. This content will give users the tools to read and navigate any Entity list page.

‌
~35 min.
Navigate Entity detail pages
Module

Detail pages allow users to see information about a single entity record, such as a single Shot or Playlist. This content gives users the tools to read and navigate any Entity detail page.

‌
~35 min.
Organize data on an Entity page
Module

Whether users are working within Entity list or detail pages, there are many ways to organize data within ShotGrid. Learn how to view only a certain number of results on a page, show page summaries, edit what fields are visible on a page, sort and group fields, and filter records with the filter panel.

‌
~25 min.
Format data on an Entity page
Module

Once data is organized, formatting can be used to further isolate information on an Entity page and to make a page more appealing. This content will introduce header and cell colors, naming and sizing, and conditional formatting rules.
