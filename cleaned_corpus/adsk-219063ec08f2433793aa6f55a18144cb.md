# About Generating Associative Spaces Workflow

Associative spaces are generated from building geometry in the drawing and are updated if it changes. Updates can be triggered automatically or manually.

| Task | Comment | |
|---|---|---|
| Create the floor plan or open a drawing | Use walls or other building objects to create the floor plan. If you are working in a structured AutoCAD Architecture 2022 toolset project, open a construct from the Project Navigator. |
|
| Click the type of space to add | Select a space tool to generate a space. You can specify that a tag is added as the space is created on the Properties palette. | |
| Create a space | Specify Generate on the Properties palette, and then specify a point within bounding geometry, or type G to generate spaces within all valid boundaries. If you change your bounding geometry and the space does not update, check the space settings. |
