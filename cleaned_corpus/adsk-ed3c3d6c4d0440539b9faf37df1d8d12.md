# Autodesk License File Parser | Administrator Installation

The License File Parser reads your license file and shows its contents in a table format. The parser is useful for getting license details when you specify parameters in an options file.

- Go to
[the License File Parser](https://www.licenseparser.com). - Click Choose File to browse to your license file.
- Select the license file and click Open.
- Click Submit License File.

The parser displays details of your product licenses.
