# Settings for Autodesk Desktop App | Administrator Installation

When you choose a product during deployment using the classic installer, you can enable, disable, or configure the Autodesk desktop app for computers in your deployment.

## Enable or disable installation of the Autodesk desktop app

When you choose a product from the classic installer, the Autodesk desktop app is selected by default.

- Deselect it if you don’t want to install the Autodesk desktop app with the product.

## Specify the Autodesk desktop app settings for network devices

- From a locally running instance of the Autodesk desktop app, go to the Settings dialog box.
- Select the settings that you want to deploy to all managed devices in your network.
- Click Export, enter a descriptive file name, and specify where to save the settings INI file.
- Open the configuration panel under the Autodesk desktop app in your deployment installer.
- Select Advanced and browse to the location of the settingsINI file.
- Add the file to your administrative image.
