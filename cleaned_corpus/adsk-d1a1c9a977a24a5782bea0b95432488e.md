# Modify a deployment with the classic installer | Administrator Installation

For products that create a deployment with the classic installer, follow these instructions to change a deployment, use an existing deployment as a template, or replace a deployment.

**Note: **To modify a deployment created with Autodesk Account, see [Modify a deployment from Autodesk Account](/support/download-install/admins/create-deployments/create-deployments-from-autodesk-account/modify-a-deployment-from-autodesk-account).

## To modify an existing deployment

What you can modify varies according to the settings you chose when you created the administrative image in the original deployment.

- Open the shared network folder that contains the deployment.
- In the Tools folder, double-click the Create & Modify Deployment shortcut.
- Click through the deployment configuration pages and select the required options.
- Click Create Deployment.
