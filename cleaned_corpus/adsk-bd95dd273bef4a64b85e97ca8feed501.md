# Character Generator is no longer available for purchase as a standalone product

Starting August 7, 2021, standalone subscriptions of Character Generator software will no longer be available for purchase. You can continue to gain access to Character Generator as part of a Maya, Maya LT, 3ds Max, MotionBuilder, Mudbox software, and Media & Entertainment Collection subscription entitlement.

## Character Generator features

With powerful 3D design and animation tools, Character Generator offers artists a web-based laboratory to create fully rigged 3D characters for animation packages and game engines.

- Source character blending –Blend two source characters from a library to create your own unique 3D character.

- Customize character features –Customize and fine-tune your 3D character from a catalog of over 100 body types, outfits, hairstyles, and physical attributes in a few simple steps.

- Select animation-ready characters –Choose from a selection of character rigged models for use in animation packages and game engines.
