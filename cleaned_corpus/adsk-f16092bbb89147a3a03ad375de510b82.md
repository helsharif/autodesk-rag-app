# Download Previous Autodesk Software Versions | Individual Installation

Some previously released versions of Autodesk software are available as a benefit when you subscribe to a current version of Autodesk software.

## Which previous versions can you download?

Flex users and customers with an active subscription. See

[Eligible previous versions for subscribers](https://knowledge.autodesk.com/customer-service/account-management/users-software/previous-versions/eligible-previous-version-subscription).Education customers. Go to the

[Education community](http://www.autodesk.com/education/home).

## To download a previous version of Autodesk software

You don't have to uninstall the current version to download and install a previous version.

Students and educators: If you have an education license, go to the [Education community](http://www.autodesk.com/education/home).

### Product versions up to three years back

- Sign in to Autodesk Account at
[manage.autodesk.com](https://manage.autodesk.com). - Under All Products and Services, find your product.
- In the product tile, click the current version and select a previous version.
- Download your product.

### Product versions four to five years back

- Sign in to Autodesk Account at
[manage.autodesk.com](https://nam11.safelinks.protection.outlook.com/?url=https%3A%2F%2Fmanage.autodesk.com%2F&data=05%7C01%7Ctrackr%40autodesk.com%7Cbfe7e1d4c74f42a36f2e08db6877b71c%7C67bff79e7f914433a8e5c9252d2ddc1d%7C0%7C0%7C638218635595429708%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C3000%7C%7C%7C&sdata=5lRpK29XjSH3fANfRgz6EcFfiIZ4dEL%2By43jdFQds%2F0%3D&reserved=0). - Click the Autodesk Assistant icon (question mark in bottom right corner).

Note that support, including technical support, software updates, and hotfixes, is only provided for previous releases up to three years back.
