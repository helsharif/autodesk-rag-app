# Autodesk Customer Success Hub

# BIM Collaborate Products Quick Start Guide

Welcome to the Quick Start Guide for BIM Collaborate and BIM Collaborate Pro. It describes the functionality and services associated with both offerings. It also covers BIM 360, the Autodesk Construction Cloud, and the management of projects and users.

## Course modules

Learn about the Autodesk Construction Cloud (ACC) and BIM 360 platforms that support BIM Collaborate and BIM Collaborate Pro.

Learn the key concepts of BIM Collaborate and BIM Collaborate Pro. Topics include target persona and workflows for each along with differences between BIM Collaborate and BIM Collaborate Pro.

The user interface has all the tools you need to work on a drawing.

Learn how to manage your account and projects using the Account Admin and Project Admin modules.

Discover the tasks geared toward architects in BIM Collaborate and BIM Collaborate Pro.

Learn about using Collaboration for Civil 3D and Collaboration for Plant 3D in BIM Collaborate Pro.

Learn about the tools for reviewing and sharing data between project collaborators.

Understand how to coordinate your design models using clash detection and issues.

View metrics and dashboards to better understand project and account performance.
