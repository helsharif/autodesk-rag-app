# Create Standalone Autodesk Deployment | Administrator Installation

You can deploy a product with a user license (based on a user’s Autodesk Account profile) or standalone device license (based on an assigned serial number). Tell users to navigate to the administrative image shortcut created for the deployment on the network share. When users double-click the shortcut or batch file, they install products on their computers.

For detailed information about products with named user and standalone licenses, see [Software installation for individuals](/support/download-install/individuals).
