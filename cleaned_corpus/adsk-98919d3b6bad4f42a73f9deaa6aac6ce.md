# Autodesk e-Learning Resources for Distance Learning | Autodesk

## Autodesk software options for remote learning

With the Autodesk Education plan, [students](/education/students) and [educators](/education/educators) have single-user access to desktop, cloud, web and mobile products and services, which can be installed on up to [three personal devices](https://www.autodesk.com/company/terms-of-use/en/general-terms/educational-licensees-additional-terms) for use at home or school.

### Desktop

Browse and access available products through the [Education Community](/education/edu-software/overview).

### Cloud-based

-
Fusion 360 cloud-based 3D CAD/CAM software has everything you need to unify design, engineering and collaboration with your peers from different locations. It works on both Mac and PC and only requires the installation of a light desktop client. Get it

[here](https://www.autodesk.com/education/edu-software/overview?sorting=featured&page=1&filters=individual&search=fusion). -
Fusion Team’s administrative features support classroom collaboration with permissions, file sharing, versioning, mark-ups and comments - all in one place. Find more information on how to use Fusion Team in an educational setting

[here](https://knowledge.autodesk.com/support/fusion-360/learn-explore/caas/simplecontent/content/getting-started-fusion-team-for-academia.html).

### Browser-based and Chromebooks

Students who are unable to install Autodesk software can use our online, browser-based tools, web apps, and mobile products. These products offer quick and easy access without the need to install software. These are great options for students with Chromebooks or mobile devices.

-
Fusion 360 is

[now available on Chromebooks](https://fusion.online.autodesk.com/)to anyone with an Autodesk account and an educational subscription. Read the[FAQs](https://knowledge.autodesk.com/support/fusion-360/learn-explore/caas/simplecontent/content/fusion-360-chromebooks-faq.html),[access guide](https://damassets.autodesk.net/content/dam/autodesk/www/pdfs/fusion_360-for-chromebook-access-guide-en.pdf)or[Contact Us](https://www.autodesk.com/education/support/edu-contact-us-form)for more information. -
The Autodesk Tinkercad app is free and easy-to-use for 3D design, electronics, and coding. Get started at

[www.tinkercad.com](http://www.tinkercad.com/)or download the Tinkercad app.

### Web apps

-
Newly available to students and educators, the Autodesk AutoCAD web app enables you to

[access AutoCAD](https://damassets.autodesk.net/content/dam/autodesk/www/pdfs/AutoCAD-web-app-access-instructions-en.pdf)from any computer to view, create, and edit AutoCAD drawings in a simplified but familiar interface.[Sign in](https://web.autocad.com/)with your Autodesk Education account to get started. -
FormIt intuitive 3D sketching tool is ideal for the conceptual design phase of architectural modeling. Go to the

[free FormIt app](https://app.formit.autodesk.com/)to access basic features.[Learn more](/archive-glfy23/formit/overview)
